# Zorteok (Lua Disassembler)

## Zorteok - Smart Lua Disassembly Engine

### Features
- Disassembles compiled lua scripts (BO2, 3, 4)
- Smart disassembly, tracing functions, arguments, loops
- Drag and Drop to disassemble a file
